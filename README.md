# EximEx - Global Import Export Trading Platform

EximEx is a comprehensive global trading platform designed for seamless import-export operations. Built with React.js and modern web technologies, it provides real-time market insights, secure transactions, and access to worldwide trading networks.

## Features

- **Global Trading Platform**: Connect with importers and exporters worldwide
- **Real-time Market Insights**: Get up-to-date market data and trends
- **Secure Transactions**: Safe and reliable payment processing
- **Document Management**: Handle all trade documentation digitally
- **Logistics Integration**: Streamlined shipping and logistics coordination
- **Multi-currency Support**: Trade in multiple currencies
- **Responsive Design**: Works seamlessly on all devices

## Technology Stack

- **Frontend**: React.js with TypeScript
- **Styling**: Tailwind CSS
- **State Management**: Redux Toolkit
- **Routing**: React Router
- **Build Tool**: Vite
- **UI Components**: Ant Design, Material-UI

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn package manager

### Installation

1. Clone the repository

```bash
git clone <repository-url>
cd Eximex-frontend
```

2. Install dependencies

```bash
npm install
```

3. Start the development server

```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
- `npm run format` - Format code with Prettier
